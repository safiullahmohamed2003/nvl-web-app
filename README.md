# NVL Education App

NVL (Nurturer Venue for Learning) - A complete education platform for institutes.

## ✨ Features

✅ User Authentication
✅ Student Dashboard
✅ Course Management & Enrollment
✅ Assignments & Submissions
✅ Grades & Performance Tracking
✅ Real-time Messaging
✅ Attendance Tracking
✅ Payment Integration
✅ Video Streaming

## 🛠️ Tech Stack

**Frontend:** React.js + Material-UI + Vite
**Backend:** Node.js + Express
**Database:** MongoDB
**Authentication:** Firebase
**Deployment:** Vercel (Frontend) + Render (Backend)

## 🚀 Quick Start

### Frontend (Local Development)
```bash
cd frontend
npm install
npm run dev
```
Frontend runs at: http://localhost:5173

### Backend (Local Development)
```bash
cd backend
npm install
npm run dev
```
Backend runs at: http://localhost:5000

## 📊 Live Deployment URLs

- **Backend:** https://nvl-api.onrender.com
